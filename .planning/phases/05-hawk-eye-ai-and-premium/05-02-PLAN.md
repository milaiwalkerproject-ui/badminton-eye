Create Configuration.storekit as a StoreKit Configuration file (JSON format) with:
- Subscription group "premium" containing:
  - "hawkeye_monthly" at $4.99/month, display name "Hawk Eye Monthly"
  - "hawkeye_yearly" at $29.99/year, display name "Hawk Eye Yearly"
- Use StoreKit Configuration file format version 3

Create SubscriptionManager.swift as @Observable singleton (matching AuthManager pattern):
- `static let shared = SubscriptionManager()`
- Properties:
  - `isPremium: Bool = false` (the single gating flag used everywhere)
  - `currentSubscription: StoreKit.Product.SubscriptionInfo.Status?`
  - `availableProducts: [Product] = []`
  - Private: `productIDs = ["hawkeye_monthly", "hawkeye_yearly"]`
  - Private: `updateListenerTask: Task<Void, Error>?`

- init(): Start `updateListenerTask = listenForTransactions()`; call `Task { await loadProducts(); await updateSubscriptionStatus() }`

- `func loadProducts() async`: Load products via `Product.products(for: productIDs)`, sort by price ascending, assign to availableProducts

- `func purchase(_ product: Product) async throws -> Transaction?`:
  - Call `product.purchase()` (no UI context needed for iOS 17+)
  - Switch on result: .success(let verification) -> verify with `checkVerified(verification)`, call `await updateSubscriptionStatus()`, return transaction
  - .userCancelled -> return nil
  - .pending -> return nil

- `func restorePurchases() async`: Call `AppStore.sync()`, then `await updateSubscriptionStatus()`

- `func updateSubscriptionStatus() async`:
  - Iterate `Transaction.currentEntitlements` using `for await result in Transaction.currentEntitlements`
  - Check if any verified transaction has productID in productIDs and is not expired/revoked
  - Set `isPremium = true` if active subscription found, `false` otherwise

- Private `func listenForTransactions() -> Task<Void, Error>`: Return Task.detached that loops `Transaction.updates`, verifies each, calls `await updateSubscriptionStatus()`

- Private `func checkVerified<T>(_ result: VerificationResult<T>) throws -> T`: Unwrap .verified case, throw for .unverified

In BadmintonEyeApp.swift, add a stored property `@State private var subscriptionManager = SubscriptionManager.shared` so it stays alive for the app lifetime. No other changes needed since SubscriptionManager.shared self-initializes.

Add SubscriptionManager.swift and Configuration.storekit to project.pbxproj. The .storekit file should be added as a resource (PBXResourcesBuildPhase), not a source.
